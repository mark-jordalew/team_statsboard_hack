All dashboard pages should be composed from these classes.
Check out the templates if you want to see how they can be used together and you can copy the template code for an easy start.
NOTE: Make sure to add the dashboard.css stylesheet to the top of your page.
.page Wrap the page in this class in order to control main padding and colors
.page--inverse A dark themed dashboard page with white text (grafana style)
.page--centered Adds spacing around the edges of the dashboard page
.grid Use this to define a grid that fills the page width and height
.row Used inside a grid
.row--flex A row that takes up as much vertical space as possible (useful for iframes)
.row--separated A row that gets spacing before
.col Used inside a row to provide blocks of the same width
.title A large page title for use in the top of the page
.heading A heading for a section of a page
.text Free longform text
